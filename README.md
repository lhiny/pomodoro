# pomodoro
Pomodoro method timer, in CLI form.

## Installing
Make sure you have libncurses installed. Run:

```
$ git clone https://github.com/lhiny/pomodoro.git && cd pomodoro
$ sudo make install
```

Alternatively, to uninstall:

```
$ sudo make uninstall
```

Note that you'll stil need the Makefile in order to uninstall the program.

## Usage

```
$ pomodoro
```

Press 'enter' to start/pause the timer, 'q' to exit.
